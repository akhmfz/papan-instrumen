# 📈 Papan Instrumen
> All-in-One Fundamental Dashboard for the Indonesian Stock Market built with Pine Script® v6.

---
## 🚧 Status
**Phase 0 — Foundation**
This project is currently under active development.
The goal is to build a professional fundamental analysis dashboard specifically designed for the Indonesian Stock Exchange (IDX).

---
## 🎯 Vision
To provide Indonesian investors with an intuitive, reliable, and comprehensive analysis dashboard directly on TradingView.

---
## 👥 Target Users
- Value Investors
- Long-term Investors
- Swing Traders
- Capital Market Study Groups
- Indonesian Retail Investors

---
## 🇮🇩 Focus Market
Indonesia Stock Exchange (IDX)
This project is designed specifically for the Indonesian market and may not be suitable for other exchanges without adjustments.
