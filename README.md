# Overview
**Saham: Papan Instrumen By. Akhmfz** merupakan dashboard analisis fundamental yang dikembangkan khusus untuk **Pasar Modal Indonesia (IDX)** menggunakan **TradingView Pine Script v6**.
Proyek ini bertujuan menyederhanakan proses analisis fundamental sehingga investor Indonesia dapat mengambil keputusan investasi yang lebih objektif, cepat, dan mudah dipahami langsung dari chart TradingView.
Berbeda dengan dashboard fundamental pada umumnya yang mengadopsi pendekatan pasar Amerika Serikat, proyek ini dibangun dengan metodologi yang disesuaikan dengan karakteristik emiten Indonesia.
